My Example of Calculator in Python

My calculator program demonstrates the principles of object oriented programming such as Encapsulation, Inheritance, Polymorphism, and Abstraction

Inheritance

In the calculator program, i used Inheritance to create my addition, subtraction, multiplication, and divison classes. they inherit the calcuations class.

class Addition(Calculation):
"""The addition class has one method inherit Calculation class"""
def get_result(self):
"""encapsulation method"""
return reduce(lambda x, y:x + y, self.values)

as shown above in the code, it on the first line when i declared the class with the parameter Calculation, that means i inherited all of Calculation class. so my addition class is a part of this class

Encapsulation

In my program, I used encapsulation in my addition, subctraction, etc. classes for my get_result() method. this method calculates the value of the calcuation given. it is encapsulated because only the object created has access to this method.

class Subtraction(Calculation):
"""The Subtraction class has one method inherit Calculation class"""
def get_result(self):
"""encapsulation method"""
return reduce(lambda x, y:x - y, self.values)

Polymorphism

in my program, i have considered Polymorphism. the idea of it that we can take parameters that are not just integers. what if the user wants to add more than one number? so i have decided that the user can give any number of paramters to the function and in my class, i will put all the paramters inside a tuple. so no matter how many paramters are given, my program does not break.

class Calculation:
"""Calculation class """
def __init__(self,values: tuple):
"""constructor for Calculation Class"""
self.values = Calculation.convert_floats(values)
@classmethod
def create(cls, values: tuple):
""" factory method"""
return cls(values)
@staticmethod
def convert_floats(values):
""" converts input from ints to floats and puts them in a lst"""
lst = []
for item in values:
lst.append(float(item))
return lst

as shown here, in conver_floats method i concatinate them in floads and return them as a lst of floats
